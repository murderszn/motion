## Usage Notes

### Attribution

- **CC0 / Public Domain**: No attribution required
- **MIT / Apache**: Include license file or text
- **CC BY 4.0**: Credit the creator (e.g., "Icons by Lucide")
- **Unsplash / Pexels**: Appreciated but not required

### Optimization

- Compress images with [Squoosh](https://squoosh.app/) or [Sharp](https://sharp.pixelplumbing.com/)
- Use WebP/AVIF for photos, SVG for icons/illustrations
- Lazy-load off-screen images: `loading="lazy"`
- Serve responsive images with `srcset` and `sizes`

### WebGL Texture Tips

- Use power-of-two dimensions (256, 512, 1024) for mipmapping
- Keep textures <2MB for mobile performance
- Use `CLAMP_TO_EDGE` wrapping for background textures
- `LINEAR` filtering for smooth, `NEAREST` for pixel-art style
