Workshop Agenda

A breakdown of the 3-hour intensive technical curriculum.

09:00 - 09:45

AI Capabilities & Foundations

Exploring state-of-the-art LLM capabilities, advanced prompt engineering, context window mechanics, and vector embedding pipelines. Establishing base models and parameters using Claude and Codex.

09:45 - 10:30

Agentic Coding & Workspace Automation

Implementing autonomous code generation loops, real-time syntax checking, self-correcting logic runtimes, and local workspace file patching using Codex APIs and custom workspace tool calls.

10:30 - 11:15

AI Skills, Hooks & Stateful Workflows

Designing reusable agent skill catalogs, dynamic tool hooking protocols, stateful multi-step workflows, and asynchronous system trigger hooks to orchestrate complex execution patterns.

11:15 - 12:00

Full-Scale AI Systems Orchestration

Building end-to-end multi-agent orchestration frameworks. Deploying collaborative networks of Claude and Codex instances to solve complex, parallel software development workflows.
