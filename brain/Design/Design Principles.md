# Design Principles

The full ten principles extracted from [[Design Philosophy]].

1. **Paint with Math, Not Pixels** — shaders describe fields, not drawings
2. **Design the Output Space** — define possibility, not specific outcomes
3. **The Seamless Loop is a Contract** — last frame = first frame
4. **Noise is Your Generative Engine** — FBM, domain warping, ridged noise
5. **Constraints Breed Creativity** — 4-digit seed, 7 sliders, 4 stops
6. **Immediate Feedback is Non-Negotiable** — 60fps, no save/preview
7. **Add Analog Imperfections** — grain, vignette, rim lights
8. **Think in Systems, Not Frames** — procedures that generate families
9. **Export is a Feature, Not an Afterthought** — zero-friction sharing
10. **Audio is a First-Class Input (Future)** — sound-driven parameters
