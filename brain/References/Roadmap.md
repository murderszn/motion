# Roadmap

> From shader playground → full moving-image studio.

## Current: Phase 0 — Foundation ✅

- 14 procedural shader presets
- Seed-based determinism
- 4-stop palette system (11 palettes)
- 7 parametric sliders
- PNG / WebM / GIF export
- Seamless loop (circular phase)
- xterm.js terminal
- VSCode-style layout
- Theme system
- **TypeScript + Vite conversion** — modular codebase with typed state, UI modules, and shader hot-swap
- Command palette (Cmd+K)
- Live shader editor
- Text overlay system
- Project save/load (`.lumen` format)

## Next: Phase 1 — The Studio Shell

- Settings persistence (localStorage)
- Plugin API v0
- Export queue

## Phase 2 — Timeline & Layers

- Layer system (shader, text, image, video)
- Keyframe engine with bezier interpolation
- Pre-compositions
- Blending modes
- Adjustment layers

## Phase 3 — Node Graph

- Per-layer node editor
- 50+ built-in nodes (color, blur, distort, etc.)
- Shader node (any `.glsl`)
- Expression editor (JS on parameters)
- Preset browser

## Phase 4 — Platform

- Plugin marketplace
- Real-time collaboration
- Data-driven animation (CSV/JSON)
- Interactive export (Rive-style)
- Audio-reactive
- AI utility nodes
