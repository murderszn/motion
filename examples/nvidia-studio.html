Depth of Field & Separable Blurs
Circle of Confusion (CoC)
// GPU Gems 3, Chapter 28
float coc = abs(depth - uFocus) * uAperture;

The focus distance matches uFocus. The blur radius scales linearly relative to the distance from the focus plane.
Separable Gaussian Blur
// GPU Gems 2, Chapter 40
Instead of N² texture samples in a single 2D pass, separable filters run a Horizontal pass, then a Vertical pass.

Samples count decreases from 81 fetches (9x9) to 18 fetches (9+9).
Velocity Vector Motion Blur
// GPU Gems 3, Chapter 27
Accumulates pixels along the velocity vector projection-space delta:
v = p_current - p_previous;
Offset = velocity * exposure;
